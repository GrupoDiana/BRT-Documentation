# Readers

There are two file loaders integrated into the library, one for loading Spatially Oriented Format for Acoustics files <a href="https://www.sofaconventions.org/mediawiki/index.php/SOFA_(Spatially_Oriented_Format_for_Acoustics)" target="_blank">(SOFA)</a> and another for loading OBJ files.

- [SOFA Reader](./sofa-reader.md): The SOFA Reader class is designed to handle the reading and parsing of audio resource data from files in the SOFA format (different data types and conventions are supported).
- [OBJ Reader](./obj-reader.md): The OBJ loader allows you to load the geometry and acoustic parameters of rooms.
